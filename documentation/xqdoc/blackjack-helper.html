Blackjack/Helper

Module Description

Module Name: Blackjack Helper This module is used for helper functions like generating a Deck or a random number. The generated random number determines Ids or draws random cards out of the deck.

  • Author: Moritz Issig, Patryk Brzoza, Fami Mahmud
  • Version: 1.0
  • See: e.g. chapter helper in the documentation
